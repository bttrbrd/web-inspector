# MySQL Web Inspector

## ФИО: Жигалова Виктория Александровна
## Назначенная технология: MySQLi (PHP)

## Описание проекта
Упрощенный веб-интерфейс для просмотра и исследования баз данных MySQL, аналогичный phpMyAdmin, но с упрощенным функционалом.

## Функциональность
- Просмотр списка баз данных
- Просмотр списка таблиц в выбранной базе данных
- Просмотр структуры таблиц (поля, типы, ключи и т.д.)
- Просмотр данных таблиц с постраничной навигацией
- Обработка ошибок подключения и доступа

## Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Расширение MySQLi для PHP

## Установка и запуск

1. **Скачайте файлы проекта** в папку вашего веб-сервера (например, `htdocs/mysql-inspector/`)

2. **Настройте подключение к БД** в файле `config.php`:
```php
define('DB_HOST', 'localhost');      // Хост MySQL
define('DB_USER', 'root');           // Пользователь MySQL
define('DB_PASSWORD', '');           // Пароль пользователя
define('DB_CHARSET', 'utf8mb4');     // Кодировка
